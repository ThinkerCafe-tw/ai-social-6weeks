# 第一週：建立基礎 - 從混亂到第一篇貼文

> 核心目標：發出第一篇用 AI 協作完成的貼文

---

## 本週課程安排

### 週一（1 小時）：2W1H 框架 + AI 溝通基礎

**課程目標：**
- 理解自媒體經營的核心思維：你為誰做？做什麼？怎麼做？
- 學會和 AI 正確溝通的方式
- 完成第一篇貼文的創作

**教學流程：**

#### 1. 開場引導（10 分鐘）
- 了解沛綺目前的自媒體經營狀況
- 討論她遇到的具體困難
- 設定本週學習目標

#### 2. 2W1H 框架講解（20 分鐘）

**WHO - 你的受眾是誰？**

引導問題：
- 你現在的客戶是什麼樣的人？
- 他們的年齡層、職業、生活狀態？
- 他們為什麼會需要你的服務？
- 你理想的客戶是誰？

**WHAT - 你要給他們什麼？**

引導問題：
- 他們有什麼痛點或需求？
- 你能為他們解決什麼問題？
- 你想傳遞什麼價值？
- 什麼內容會讓他們想追蹤你？

**HOW - 你要怎麼做？**

引導問題：
- 你的內容風格是什麼？
- 你想給人什麼樣的感覺？
- 你的發文頻率和平台？
- 你的獨特之處在哪裡？

#### 3. AI 溝通示範（20 分鐘）

**錯誤示範：**
```
ChatGPT，幫我寫一篇關於原生家庭的貼文。
```

**為什麼不好？**
- 太籠統，AI 不知道你要什麼角度
- 沒有給背景資訊
- 產出的內容會很空泛、不夠個人化

**正確示範：**
```
我是一位身心靈療癒師，主要服務對象是 25-40 歲在職場感到壓力、
同時有原生家庭議題困擾的都市女性。

我想寫一篇 Instagram 貼文，主題是「原生家庭對我們的影響」。

我希望這篇文章：
1. 開頭用一個具體的生活場景引起共鳴
2. 中間點出原生家庭如何影響我們的人際關係
3. 結尾給予溫暖的鼓勵和行動建議
4. 語氣要溫暖但不矯情，專業但不距離感
5. 字數約 200-250 字

請幫我生成這篇貼文。
```

**為什麼這樣好？**
- 清楚說明你的身份和受眾
- 具體描述你要的內容結構
- 定義了語氣和風格
- 給了明確的字數範圍

#### 4. 實作練習（10 分鐘）

讓沛綺選擇一個她這週想發的主題，現場帶她用正確的方式跟 ChatGPT 對話，完成一篇貼文。

**作業：**
- 把這篇貼文發布到 Instagram 或 Threads
- 截圖給我看
- 記錄發布後的感受和觀察

### 週三（1 小時）：檢討第一篇貼文 + 建立風格模板

**課程目標：**
- 分析第一篇貼文的優缺點
- 定義個人風格
- 建立可重複使用的 prompt 模板

**教學流程：**

#### 1. 貼文檢討（15 分鐘）

**檢視要點：**
- 這篇文章聽起來像你嗎？
- 哪些部分你很滿意？
- 哪些部分覺得可以更好？
- 發布後有什麼反應？

**改進方向：**
- 如果太商業化 → 加入更多個人經驗和溫度
- 如果太文青 → 加入更多實用建議
- 如果太平淡 → 用更具體的場景和故事

#### 2. 風格定義工作坊（25 分鐘）

**找參考對象：**

讓沛綺列出 3-5 個她喜歡的創作者，分析他們的風格：
- 他們的貼文給你什麼感覺？
- 你喜歡他們哪些地方？
- 你想學習他們的什麼特質？

**定義你的風格關鍵詞：**

從以下維度思考：
- 語氣：溫暖的、專業的、陪伴式的、知識型的
- 結構：故事開場、痛點提問、案例分享
- 長度：簡短精煉（150 字內）、適中（200-300 字）、深度長文（400+ 字）
- 互動：提問式、分享式、教學式

**示範：沛綺的風格可能是：**
- 溫暖陪伴型：像朋友聊天，不說教
- 具體場景導向：用生活化的例子引起共鳴
- 實用行動導向：不只講道理，給具體方法
- 適中長度：200-300 字，適合 IG 閱讀

#### 3. 建立 Prompt 模板（15 分鐘）

根據定義好的風格，建立一個可重複使用的模板：

```
【沛綺的 IG 貼文 Prompt 模板】

我是一位身心靈療癒師，主要幫助 [受眾描述] 解決 [核心問題]。

我想寫一篇 Instagram 貼文，主題是「[本次主題]」。

內容結構：
1. 開頭：用一個具體的生活場景或感受引起共鳴
2. 中段：[依主題調整，可能是：說明原因、分享觀點、案例分析]
3. 結尾：給予溫暖的鼓勵和 1-2 個具體可行的行動建議

風格要求：
- 語氣：溫暖陪伴，像朋友聊天，不說教
- 用詞：生活化，避免艱深術語
- 舉例：具體場景，讓人有畫面感
- 長度：200-300 字

請幫我生成這篇貼文。
```

**使用方式：**
- 每次要寫貼文時，複製這個模板
- 只需要填入 [本次主題] 和調整中段結構
- 其他風格設定都自動套用

#### 4. 實作練習（5 分鐘）

用新模板生成 3 篇不同主題的貼文（不用發布，只是練習）。

**作業：**
- 用新模板寫 3 篇貼文
- 自己讀一遍，感受是否像自己的風格
- 下週一帶來討論

## 常見問題 Q&A

**Q1: 用 AI 寫的文章會不會被看出來？**

A: 重點不是「不被看出來」，而是「聽起來像你」。如果你用對的方式引導 AI，加上你的個人經驗和觀點，讀者感受到的會是「你的想法被清楚表達」，而不是「AI 生成的空話」。

**Q2: 每次都要打這麼長的 prompt 嗎？**

A: 第一次建立模板時需要，之後你只需要複製模板、填入主題就好。熟練後，5 分鐘就能生成一篇貼文。

**Q3: 如果生成的內容我不喜歡怎麼辦？**

A: 告訴 AI 你不喜歡哪裡，讓它修改。例如：「開頭太平淡，請改用更具體的場景」或「結尾太說教，請改得更溫暖」。

**Q4: 需要每篇都用 AI 嗎？**

A: 不需要。AI 是工具，不是替代品。有時候你靈感來了，直接寫可能更快。但當你卡住、沒靈感、或想要不同角度時，AI 就是很好的協作夥伴。
