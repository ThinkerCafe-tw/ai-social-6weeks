## 關鍵成功要素提醒

### 心態調整

**記住這些重點**：
```
1. 她不是新手，是要升級的使用者
   → 不要從零教起
   → 直接示範高品質方法
   → 肯定她已有的能力

2. 她可能帶著挫折感
   → 先肯定她的努力
   → 重新詮釋「失敗」
   → 建立「我可以」的信心

3. 她需要立即看到改變
   → 第一堂課就要有成果
   → 對比要明顯
   → 讓她驚嘆「原來可以這樣」

4. 她需要持續鼓勵
   → 每天保持聯繫
   → 即時給予回饋
   → 慶祝小成功
```

### 教學技巧

**保持這些原則**：
```
1. 多示範，少講理論
   → 實際操作給她看
   → 讓她看到立即效果
   → 邊做邊解釋

2. 用她的案例
   → 用她的領域（脈輪、療癒）
   → 用她的客戶情境
   → 用她的真實經驗

3. 保持彈性
   → 觀察她的反應
   → 隨時調整節奏
   → 如果她跟不上，放慢
   → 如果她超前，加快

4. 建立安全感
   → 「沒有笨問題」
   → 「不會很正常」
   → 「我們一起來」
   → 「妳做得很好」
```

### 溝通話術

**常用鼓勵語**：
```
「妳做得很好！」
「這個進步很明顯！」
「妳比妳想像的更厲害！」
「看，妳已經會了！」
「這就是妳的聲音！」
「這篇寫得很棒！」
「我對妳很有信心！」
```

**常用引導語**：
```
「我們來試試看...」
「不用擔心，我們一步步來」
「先別急，我們先...」
「妳覺得這樣如何？」
「妳想要怎麼調整？」
「讓AI幫妳...」
「我們一起來...」
```

## 應急預案

### 如果學員緊張

**處理方式**：
```
1. 放慢節奏
2. 增加鼓勵
3. 降低難度
4. 多示範，少要求她操作
5. 說「我們一起來」而非「妳來試試」
```

### 如果學員跟不上

**處理方式**：
```
1. 立即停下來
2. 詢問哪裡不清楚
3. 重新示範一次
4. 簡化步驟
5. 降低目標（先求60分，不求90分）
```

### 如果學員超前

**處理方式**：
```
1. 增加進階內容
2. 加快節奏
3. 增加實作練習
4. 提供更複雜的案例
5. 提前教Week 2的內容
```

### 如果技術故障

**處理方式**：
```
1. Google Meet 故障
   → 立即切換到 Zoom
   → 學員已有備用連結

2. ChatGPT 故障
   → 切換到 Claude
   → 或用截圖範例繼續教學

3. 網路不穩
   → 改用電話通話
   → 或改約其他時間
```
