- DAY 2：回答品牌覺察問題 + 品牌自介

- DAY 7：發限量免費服務貼文

### 第5堂（1/12）學到的黃金公式
**Hook → Vulnerability → Pivot → Insight**

1. **Hook（混亂場景）**：用感官細節讓讀者身歷其境
2. **Vulnerability（差點崩潰）**：承認脆弱，建立信任
3. **Pivot（掏出法寶）**：展示療癒工具
4. **Insight（體悟與CTA）**：給出可行動的建議

### 第5堂後的作業（週三前完成）
1. 用轉化器生成5篇貼文
2. 每篇都要真實的崩潰瞬間 + 真正會做的事
3. 修到滿意為止（至少修2-3次）
4. 如果有特別好的對話，分享給老師
