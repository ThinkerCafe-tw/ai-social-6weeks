## 📝 IG 貼文通用模板

```
【沛綺的 IG 貼文 Prompt 模板 v1.0】

我是一位身心靈療癒師，主要幫助 25-40 歲在職場感到壓力、
同時有原生家庭議題困擾的都市女性，找回內在平靜與力量。

我想寫一篇 Instagram 貼文，主題是「[本次主題]」。

內容結構：
1. 開頭：用一個具體的生活場景或感受引起共鳴
2. 中段：[依主題調整，可能是：說明原因、分享觀點、案例分析]
3. 結尾：給予溫暖的鼓勵和 1-2 個具體可行的行動建議

風格要求：
- 語氣：溫暖陪伴，像朋友聊天，不說教
- 用詞：生活化，避免艱深術語
- 舉例：具體場景，讓人有畫面感
- 長度：200-300 字
- 適度使用換行，增加閱讀舒適度

請幫我生成這篇貼文。
```

**使用時機：**
- 日常發文
- 知識分享
- 個人觀點表達

**修改建議：**
- 根據當天想要的內容類型調整中段結構
- 如果想要更長或更短的文章，調整字數要求

## 📖 故事型貼文模板

```
我是一位身心靈療癒師，想分享一個真實的療癒故事（已匿名處理）。

故事背景：[簡述案例背景]
核心衝突：[主要的痛點或困擾]
轉變過程：[如何開始改變]
最終結果：[得到什麼啟發或改善]

請幫我寫成一篇 Instagram 貼文，要求：
- 用第三人稱或化名，保護個案隱私
- 聚焦在情感轉變，而非具體療程細節
- 結尾引導讀者思考：這個故事對他們有什麼啟發
- 語氣溫暖、真誠，避免過度美化或煽情
- 字數 300-400 字

請生成這篇貼文。
```

**使用時機：**
- 想分享個案故事時
- 需要用實際例子說明觀點時
- 想增加內容的真實感和共鳴時

## 💡 知識型貼文模板

```
我想寫一篇知識型的 Instagram 貼文，主題是「[知識主題]」。

目標受眾：25-40 歲都市女性，對身心靈議題感興趣但不一定有深入了解

內容要求：
1. 開頭：用一個常見的迷思或疑問開場
2. 中段：用 3-5 個要點解釋這個概念
3. 結尾：總結並給一個簡單的實踐建議

風格：
- 深入淺出，把專業知識說得清楚易懂
- 避免學術用語，用生活化的比喻
- 適度使用 emoji 或符號標記重點
- 字數 250-350 字

請生成這篇貼文。
```

**使用時機：**
- 教育性內容
- 解答常見問題
- 建立專業形象

## 🤔 互動型貼文模板

```
我想寫一篇能引發互動的 Instagram 貼文。

主題：[互動主題]

格式要求：
- 開頭：提出一個開放式問題或情境
- 中段：給 2-3 個不同的觀點或選項
- 結尾：邀請讀者在留言區分享他們的想法或經驗

風格：
- 輕鬆、不帶批判
- 讓人覺得安全可以分享
- 避免正確答案式的引導
- 字數 150-200 字（重點是引發討論，不是長篇大論）

請生成這篇貼文。
```

**使用時機：**
- 想提高互動率時
- 想了解受眾想法時
- 建立社群連結時

## 🎯 服務介紹模板

```
我想寫一篇介紹我的療癒服務的 Instagram 貼文。

服務內容：[具體服務項目]
適合對象：[誰需要這個服務]
解決問題：[能幫助解決什麼]

要求：
- 不要硬銷，而是從「痛點」切入
- 用「如果你常常...」的句型引起共鳴
- 自然地說明服務如何幫助到他們
- 結尾給預約或諮詢的行動呼籲
- 避免誇大承諾，實事求是
- 字數 200-250 字

請生成這篇貼文。
```

**使用時機：**
- 介紹新服務時
- 定期提醒服務內容時
- 想增加預約量時

## 🔄 一稿多用模板

```
我有一篇已經寫好的 Instagram 貼文（如下），
請幫我把它改寫成以下不同形式：

【原文】
[貼上原始貼文]

請改寫成：
1. Threads 短文版（100 字以內，更口語化）
2. Instagram 限時動態文字（分成 3-4 個畫面，每個畫面 1-2 句話）
3. Facebook 長文版（400-500 字，加入更多細節和例子）

保持核心訊息不變，但調整長度和語氣以適應不同平台。
```

**使用時機：**
- 想最大化一個好內容的價值時
- 需要在多平台發布時
- 時間有限但想保持活躍時

## 📝 模板使用指南

### 基本使用流程：

1. **選擇適合的模板**
   - 根據今天想發的內容類型選擇
   - 可以混合使用多個模板

2. **填入具體資訊**
   - 把 [方括號] 內的內容換成你的實際內容
   - 可以調整字數、風格等參數

3. **生成並檢視**
   - 把完整的 prompt 貼到 ChatGPT
   - 閱讀生成的內容

4. **修改優化**
   - 如果不滿意，告訴 AI 哪裡要改
   - 可以要求重寫某一段
   - 也可以自己手動調整

5. **記錄學習**
   - 記下哪些 prompt 效果好
   - 持續優化你的模板

### 進階技巧：

**技巧 1：建立個人資料庫**
- 把你的背景資訊、受眾描述存成一份文件
- 每次只需要複製貼上，不用重打

**技巧 2：模板組合**
- 可以把多個模板的優點結合
- 例如：知識型 + 互動型 = 教學式提問

**技巧 3：持續優化**
- 每次使用後記錄效果
- 定期更新模板內容
- 加入新發現的有效元素
