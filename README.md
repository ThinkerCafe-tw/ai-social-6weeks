# AI 自媒體系統建立課程 - 六週陪跑計畫

> 由 Cruz Tang (@tangcruzz) 開發，ThinkerCafe 一對一專屬教學

## 🎯 這是什麼？

這是一套專為**身心靈產業自媒體經營者**設計的 AI 協作系統建立課程。

六週時間，從零開始建立完整的內容創作、發布、數據追蹤系統，讓你從「想到什麼發什麼」變成「有系統地持續產出優質內容」。

**適合對象：**
- 有想法但不知道怎麼寫成吸引人貼文的創作者
- 想用 AI 提升自媒體經營效率的自由工作者
- 希望建立個人品牌和自動化系統的服務業者
- 對 AI 協作感興趣但不知從何開始的初學者

---

## 📚 課程架構 (6 週)

### 第一週：建立基礎 - 從混亂到第一篇貼文
- 2W1H 框架與 AI 溝通基礎
- 個人風格定義與 prompt 模板建立
- **成果：**發出第一篇 AI 協作貼文

### 第二週：批量思考 - 建立內容規劃系統
- 內容分類與主題庫建立
- AI 圖片生成基礎（DALL-E）
- **成果：**一個月內容日曆 + Google Sheet 管理系統

### 第三週：執行優化 - 建立穩定的發文節奏
- 數據分析與內容優化
- 一稿多用技巧
- **成果：**專屬發文 SOP 檢查清單

### 第四週：自動化引流 - 將互動轉化為實際聯繫
- Instagram + ManyChat 自動私訊設定
- 引流關鍵字設計與數據收集
- **成果：**實際運作的 Instagram 自動引流機制

### 第五週：品牌建立 - 從社群到個人網站
- GitHub Pages 個人展示網站建立
- 設計從社群到聯繫的完整流程
- **成果：**個人品牌展示網站 + 完整的轉換路徑設計

### 第六週：系統整合 - 建立長期可用的工具包
- AI 工具包整理與系統化
- 未來成長規劃
- **成果：**完整的個人 AI 工具包

## 🎓 課程資訊

**時間安排：**
- 每週一、三晚上 8:30-9:30
- 每次 1 小時，共 12 小時正式課程
- 週間持續陪伴與任務追蹤（約 6 小時）
- 總計 18 小時完整教學

**週間陪伴機制：**
- 每天檢查學習進度和作業完成狀況
- 給予具體回饋和改進建議
- 追蹤實際發文和數據表現
- 即時解答遇到的問題
- 不只是上課，更是帶著你成長

- ✅ 18 小時完整教學

- ✅ 個人 AI 工具包

- ✅ 所有課程錄影檔案

## 🚀 快速開始

1. **查看課程大綱：** 
   - 打開 `index.html` 查看完整課程規劃
   - 或訪問線上版本：[待部署到 GitHub Pages]

2. **課前準備：**
   - 確認 ChatGPT 付費版可正常使用
   - 準備 Instagram / Threads 帳號
   - 建立 Google 帳號（用於 Google Sheet）
   - 註冊 GitHub 帳號（用於個人網站）

3. **開始學習：**
   - 按週次閱讀 `01-課程內容/` 中的教材
   - 完成每週的實作作業
   - 使用 `02-工具模板/` 中的模板加速創作
