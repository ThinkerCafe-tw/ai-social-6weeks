## 🎯 這是什麼

這是一套為期 **六週的 AI 自媒體實戰課程**，專注於幫助創作者、自媒體工作者、內容創業者，透過 AI 建立**可持續運作的自動化內容系統**。

**不只是學工具，而是建立一套屬於你的內容生產流程。**

---

## 💡 課程特色

### ✨ 一對一深度陪跑
- 不是錄播課程，是真人 1-on-1 教學
- 每週 2 次線上課（週一、三晚上 8:30-9:30）
- 週間每日進度追蹤與回饋
- 總計 18 小時完整教學時間

### 🎓 從系統思維出發
- 第一週就建立「你的 AI 協作框架」
- 學會如何讓 AI 理解你的風格
- 建立可複製、可迭代的工作流程

### 🚀 實戰導向
- 每週都有實際產出
- 第三週就開始經營 Threads 帳號
- 課程結束時，你會有一個運轉中的自媒體系統

### 🔄 持續優化
- 不只教技術，更教如何優化
- 建立數據追蹤機制
- 學會根據數據調整策略

| 週次 | 主題 | 核心產出 |

## 🛠️ 技術說明

> **給開發者 / 教育工作者的 Note**

這個課綱網頁是用 **單檔 HTML + React (CDN)** 打造的，包含：

### 特色功能
- ✅ 互動式確認表單（React Hooks）
- ✅ 一鍵複製功能
- ✅ 即時預覽
- ✅ 響應式設計
- ✅ 零後端依賴

### 為什麼這樣設計？
1. **快速部署**：直接 GitHub Pages，無需 build process
2. **易於維護**：單一檔案，方便修改
3. **學員友善**：直接打開就能看，不需要任何安裝

### 本地開發
```bash
# 克隆專案
git clone git@github.com:ThinkerCafe-tw/ai-social-6weeks.git

# 直接用瀏覽器打開
open index.html

# 或用 local server
python3 -m http.server 8000
# 然後訪問 http://localhost:8000
```

### 技術棧
- React 18 (CDN)
- Lucide Icons
- 純 CSS (無框架)
- 零外部依賴
