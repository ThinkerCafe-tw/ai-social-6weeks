# 互動式確認表單 - 使用說明

## ✅ 已完成功能

### 1. 互動式填寫表單
學員可以在網頁上直接填寫：
- 姓名、Email、電話
- 時間確認（下拉選單）
- 課程內容確認（下拉選單）
- 開課日期確認（下拉選單）
- 發票需求（下拉選單）
- 其他問題或備註（文字框）

### 2. 即時預覽
填寫的同時，下方會即時顯示將要複製的內容預覽

### 3. 一鍵複製
點擊「📋 複製確認表單」按鈕，會複製以下格式：

```
📋 AI 自媒體六週實戰課程 - 報名確認

👤 學員資訊
姓名：湯小明
Email：sulaxd@gmail.com
電話：0937431998

⏰ 課程確認
1. 時間：每週一、三晚上 8:30-9:30
   我的回答：可以配合

2. 課程內容：六週學習路徑是否符合期待？
   我的回答：符合

3. 開課日期：11/24（日）開始
   我的回答：可以

4. 發票需求
   我的回答：不需要發票 NT$ 9,500

💬 其他問題或備註
[無]
```

### 4. 複製成功提示
點擊後按鈕會顯示「✅ 已複製！」2 秒鐘

## 使用流程

### 學員端
1. 打開課綱頁面
2. 閱讀完整課程內容
3. 滑到「下一步」區塊
4. 填寫確認表單（選擇答案）
5. 點擊複製按鈕
6. 透過 LINE/Email/Threads 傳給你

### 你這端
1. 收到學員的確認資訊
2. 有完整的文字記錄作為合約依據
3. 確認後回覆繳費資訊：
   ```
   銀行：第一銀行（007）
   帳號：321-10-060407
   戶名：思考者咖啡有限公司
   金額：NT$ 9,500 或 NT$ 10,000
   ```
4. 收到轉帳後，寄送課前準備清單

## 優點

✅ **方便學員**：不用手動打字，選完就能複製
✅ **有記錄**：文字確認，雙方都有保障
✅ **格式統一**：所有學員的回覆都是同樣格式，方便你整理
✅ **即時預覽**：學員能看到自己填寫的內容
✅ **純前端**：不需要後端，不需要資料庫，不需要擔心安全性
